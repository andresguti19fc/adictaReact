## Run Locally

Clone the project

```bash
  git clone https://github.com/andresguti19fc/adictaReact.git
```

Go to the project directory

```bash
  cd adicta
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
